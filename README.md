# Krishi Setu: A Transparent Digital Platform Empowering Farmers

## Overview

Krishi Setu is a digital platform designed to empower farmers by eliminating middlemen exploitation and ensuring fair prices through a transparent marketplace. The platform connects farmers directly with verified buyers under government supervision.

## Key Features

- **Direct Market Access**: Farmers can list crops and connect directly with verified buyers
- **Government Supervised Tenders**: All transactions happen through government-verified tenders
- **Secure Payment System**: Safe wallet system where payments are released only after successful delivery
- **Equipment Rental**: Platform for farmers to rent out their equipment for additional income
- **Real-time Price Discovery**: Transparent bidding system ensuring best market prices
- **Simple Digital Interface**: User-friendly design for farmers with limited digital knowledge

## Technology Stack

- React 18
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd krishi-setu
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Features in Detail

### Digital Marketplace
- Farmers can list their crops with quality grades
- Buyers bid on crops through government-verified tenders
- Real-time bidding with transparent price discovery

### Equipment Rental Platform
- Farmers can list their farming equipment for rent
- Secure payment system for equipment rentals
- Tracking and verification of equipment

### Government Integration
- Complete transparency in all transactions
- Government verification at every step
- Real-time monitoring and data analytics
- Protection from middlemen exploitation
